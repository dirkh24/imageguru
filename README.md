## Features
- Encrypted user authorizaton
- Database initialization
- New user signup
- User login/logout
- User settings
- Modern user interface
- Bulma framework
- Limited custom css/js
- Easily customizable

## Setup
``` 
git clone https://github.com/dirkh24/mmwml-week5
cd mmwml-week5
pip install -r requirements.txt
python app.py
```
